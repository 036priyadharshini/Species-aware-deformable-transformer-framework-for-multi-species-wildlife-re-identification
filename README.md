# Species-Aware Deformable Transformer Framework for Multi-Species Wildlife Re-Identification

Multi-species wildlife re-identification using edge-augmented input, species-conditioned deformable transformer encoding, and hybrid ArcFace + Multi-Similarity loss — evaluated on WildlifeReID-10K across 9 species.

This repository accompanies the M.E. thesis at Government College of Technology, Coimbatore (2026) and the paper submitted to IETE Journal of Research.

## Method

The proposed framework extends [ReDeformTR](https://doi.org/10.1109/ACCESS.2024.3437168) (Li et al., IEEE Access 2024) with four enhancements:

**1. Edge-Augmented Input (4-channel)**
Each image is a 4-channel tensor: RGB + Canny edge map. The edge channel encodes structural patterns — stripes, spots, scale boundaries, contours — that are primary identity cues for many species.

**2. Species-Conditioned Transformer Encoding**
A learnable species embedding is added to the deformable attention query tokens, allowing the encoder to adapt its attention to species-specific visual traits (e.g., dorsal fin notches for whales, head scale patterns for sea turtles, rosette spots for leopards).

**3. Hybrid Loss: ArcFace + Multi-Similarity**
- **ArcFace** (Deng et al., CVPR 2019): additive angular margin for tighter inter-class boundaries.
- **Multi-Similarity Loss** (Wang et al., CVPR 2019): optimises all pairwise relationships in a batch with adaptive hard-pair mining, correlating directly with the AP retrieval metric.

**4. Species-Aware Re-Ranking**
K-reciprocal re-ranking (Zhong et al., CVPR 2017) applied within species-specific subsets, preventing cross-species neighbourhood contamination in multi-species retrieval.

**Inference:** 6-view TTA (flips, crops, colour jitter) → average embeddings → cosine distance → species-aware re-ranking.

## Dataset

[WildlifeReID-10K](https://github.com/WildlifeDatasets/wildlife-datasets) — 214,000+ images of ~10,000 individual animals across 33 species.

```bash
pip install wildlife-datasets
python -c "from wildlife_datasets import datasets; datasets.WildlifeReID10k.get_data('data/wildlifereid-10k')"
```

Place at `data/wildlifereid-10k/` so that `data/wildlifereid-10k/metadata.csv` exists.

---

## Setup

```bash
git clone https://github.com/036priyadharshini/wildlife-reid-species-transformer
cd wildlife-reid-species-transformer
pip install -r requirements.txt
```

Pre-compute edge maps before training (required):

```bash
python precompute_edges.py --data_root data/wildlifereid-10k --cache_dir edge_cache
```

---

## Training

```bash
python train.py
```

Trains for 350 epochs. Checkpoints saved to `checkpoints/` every 10 epochs; best by validation mAP saved as `checkpoints/best_model.pth`.

To resume: set `CHECKPOINT_PATH` at the top of `train.py` to your checkpoint path.

Hardware used: NVIDIA Tesla T4 (16 GB VRAM), Kaggle / Google Colab.

## Key Hyperparameters

| Parameter | Value |
|---|---|
| Embedding dimension | 2048 |
| ArcFace s / m | 64 / 0.5 |
| MS Loss alpha / beta / base | 2.0 / 50.0 / 0.5 |
| Re-ranking k1 / k2 / lambda | 20 / 6 / 0.5 |
| Batch (PK sampler) | P=8, K=4 |
| Optimizer | AdamW + cosine annealing warm restarts |
| Epochs | 350 |
